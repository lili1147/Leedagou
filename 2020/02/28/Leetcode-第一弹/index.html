11.盛最多水的容器

题目：

标签：数组

思路：

盛水最多，取决于你的最左和最右，以及中间的跨度共同决定

（1）暴力解法：从第一个开始循环遍历List,第一个即为“水箱”的最左侧。最后返回最大的区间。

最大区间的计算方法：当左边 < 右边，maxArea = 区间长度 * 左边；

​ 当左边 >右边，maxArea = 区间长度 * 右边；

​ 当左边 = 右边，maxArea = 区间长度 * 左边；

暴力解法时间复杂度过高，倒在了最后一个测试用例上，时间超过了限制。

（2）双指针法：

从两边开始查找。

  • 在每一个状态下，无论长板或短板收窄 11 格，都会导致水槽 底边宽度 -1−1：
    若向内移动短板，水槽的短板 min(h[i], h[j]) 可能变大，因此水槽面积 S(i, j)S(i,j) 可能增大。
若向内移动长板，水槽的短板 min(h[i], h[j]) 不变或变小，下个水槽的面积一定小于当前水槽面积。

​ 所以通过每次移动短板，可能得到最大的面积。通过这种方法避免了走暴力寻找的路，消去了一部分可能不变或者变小的不需要的结果。

15 三数之和

题目：

标签：数组

思路：Stages

(1) 首先对数组进行排序。例如[-4,-1,-1,0,1,2];

(2)设定三个“指针”，分别位于第一个，第二个和最后一个位置上；

(3)因为数组已经进行了排序。

  • 当最左边的nums[k]的值大于0了，说明已经不可能再出现三个值相加等于0的情况了，直接break;
  • 当k>0,且nums[k] == nums[k-1]时，出现重复值，进行跳过循环；
  • 当i<j时，s = nums[k] +nums[i] +nums[j]
    • 如果 s<0 : i值向右移，增加值，但同时若出现相同值，跳过，避免重复；
    • 如果s>0 : j值向左移， 减少值，但同时若出现相同值，跳过，避免重复；
    • 当s==0: 保存值，i,j同时移动一位，若出现重复，跳过。
